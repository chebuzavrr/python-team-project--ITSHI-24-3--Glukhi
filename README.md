# Консольний конвертер валют (Практична робота 6)

## Про проєкт
Проєкт створено в рамках вивчення командної розробки на Python. Реалізує консольний конвертер валют з актуальними курсами НБУ.

## Функціонал
- Введення сум і вибір валют (USD, EUR, GBP, PLN, UAH).
- Конвертація за фіксованими коефіцієнтами.
- Автоматичний запис усіх операцій у файл `journal.txt`.

## Розподіл ролей
Згідно з дозволом викладача, роботу виконано самостійно, але архітектуру розділено за ролями:
- **Роль А (Інтерфейс):** файл `main.py` (зчитування даних, меню).
- **Роль В (Логіка):** файл `converter_logic.py` (математика, журналювання).

## Запуск програми
Запустіть файл `main.py` у терміналі: `python main.py`
